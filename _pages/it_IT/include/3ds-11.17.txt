**Recentemente è stata rilasciata la versione del software 3DS 11.17.0**. Ecco cosa dovresti sapere:

- Se la tua console sta eseguendo *Luma 10.2.1 o superiore*, è *al 100% sicuro* eseguire l'aggiornamento. Puoi controllare la versione di Luma tenendo premuto (Select) durante l'avvio della console.
- Se la tua console ha una versione precedente di Luma, dovresti [aggiornare Luma](checking-for-cfw) prima di aggiornare la tua console alla versione 11.17.0.
- Se la tua console non possiede ancora un custom firmware, **NON aggiornare all'ultima versione di sistema**, poiché i metodi di installazione potrebbero essere più complessi.
- If your unmodified device is ALREADY on 11.17.0, this guide has been updated accordingly. Specifically, New3DS-family devices have a [free software method](homebrew-launcher-(super-skaterhax)), while Old3DS-family devices will need to use [alternate exploits](alternate-exploits) that rely on external hardware or software.
